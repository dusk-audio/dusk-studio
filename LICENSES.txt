Dusk Studio — third-party license inventory
=====================================

Dusk Studio itself is GPL-3.0-or-later (see LICENSE). This file lists every
external component bundled or vendored into the build, the license it
ships under, and the path where it lives. Update whenever a new
dependency is added so the legal surface stays auditable.

PROJECT LICENSE
---------------
Dusk Studio source code: GPL-3.0-or-later (./LICENSE)


VENDORED DEPENDENCIES (compiled into the Dusk Studio binary)
------------------------------------------------------

JUCE (audio framework)
  Version  : 8.x  (Linux pin: plugdata-team/JUCE wayland-juce8;
                   macOS/Win: upstream juce-framework/JUCE)
  License  : Dual-licensed — GPL-3.0 (used here) OR JUCE Commercial
  Path     : ../JUCE-wayland (Linux), ../JUCE (macOS/Windows)
  Upstream : https://juce.com/

Dusk Audio plugins (donor DSP cores)
  License  : GPL-3.0-or-later (matches Dusk Studio)
  Path     : ../plugins-main/plugins/  (Linux worktree)
             ../plugins/plugins/        (macOS/Windows sibling)
  Components used:
    - multi-q      (BritishEQProcessor)
    - multi-comp   (UniversalCompressor)
    - tape-echo    (TapeSaturation, legacy)
    - TapeMachine  (full tape emulation)
    - shared/AnalogEmulation (TubeEmulation, TransformerEmulation, etc.)

Catch2 (test framework — TEST BUILDS ONLY, not bundled in Dusk Studio)
  Version  : v3.5.4
  License  : Boost Software License 1.0
  Path     : build-tests/_deps/catch2-src/ (FetchContent at configure time)
  Upstream : https://github.com/catchorg/Catch2/

WHAT THIS MEANS FOR YOU
-----------------------

Self-build from source : All components compatible with GPL-3.0.
                          You can build and use Dusk Studio freely.
Paid Patreon binaries  : You are paying for support, packaging, and
                          ongoing development access. The source is
                          still GPL — your binary's compliance with the
                          GPL is unaffected by the payment.
Redistribute the binary: Allowed under GPL-3.0 terms. Include LICENSE,
                          this file, and source-code availability.
